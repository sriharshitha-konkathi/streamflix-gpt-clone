# 🎬 StreamFlix - AI-Powered Movie Streaming App

A Netflix-inspired web app with AI-powered movie search built with React, Redux, Firebase, and TMDB API.

## 🚀 Live Demo
👉 [https://streamflix-ai-gpt-clone.vercel.app/](https://streamflix-ai-gpt-clone.vercel.app/)

## ✨ Features
- 🔐 User Authentication (Sign Up / Sign In / Sign Out) with Firebase
- 🎥 Movie trailer autoplay on Browse page
- 🤖 AI-powered movie search using NVIDIA LLaMA model
- 🎬 Movie rows (Now Playing, Popular, Top Rated, Trending, Upcoming)
- 🌍 Multi-language support (English, Hindi, Telugu, Tamil, Spanish)
- 🔒 Protected routes for authenticated users
- 📱 Responsive design with Tailwind CSS
- 🗃️ Global state management with Redux Toolkit

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS
- **State Management:** Redux Toolkit
- **Authentication:** Firebase Auth
- **Database:** Firebase
- **Movie Data:** TMDB API
- **AI Search:** NVIDIA LLaMA 3.1
- **Deployment:** Vercel

## 🏃 Run Locally

```bash
git clone https://github.com/sriharshitha-konkathi/streamflix-gpt-clone.git
cd streamflix-gpt-clone
npm install
npm start
```
